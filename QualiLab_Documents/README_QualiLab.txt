Utilisation :

1. Ouvrir l'application QualiLab.
2. Aller dans le module "Dossier local".
3. Cliquer sur "Connecter un dossier".
4. Selectionner ce dossier : QualiLab_Documents.
5. Scanner le dossier pour afficher les fichiers.
6. Importer des fichiers dans le sous-dossier voulu.
7. Exporter les donnees de l'application dans 08_Exports.

Formats reconnus :

- PDF
- Images : PNG, JPG, JPEG, GIF, BMP, WEBP, SVG
- Word : DOC, DOCX, ODT, RTF
- Excel : XLS, XLSX, CSV, ODS
- Texte : TXT, JSON, XML, MD, LOG

Remarque :

L'application peut detecter et ouvrir les fichiers depuis le navigateur.
Les apercus directs sont disponibles pour PDF, images et textes.
Pour Word et Excel, l'ouverture complete depend des applications installees sur le PC.
