# QualiLab by ENNAJEH

Prototype d'application web statique pour la digitalisation du systeme de management qualite d'un laboratoire de metrologie.

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Naviguer entre les modules depuis la barre laterale.
3. Ajouter un equipement via le formulaire pour tester le comportement dynamique.

## Fonctionnalites

- tableau de bord avec indicateurs et alertes d'etalonnage
- registre des equipements avec recherche et filtre
- vues pour etalonnages, certificats, methodes, non-conformites, audits et utilisateurs
- sauvegarde locale des equipements via `localStorage`
